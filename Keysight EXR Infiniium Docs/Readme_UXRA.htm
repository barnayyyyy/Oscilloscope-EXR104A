Probe and Accessory Manuals

Oscilloscope probe and accessory manuals are found in the Keysight Probe Resource Center.

The Keysight Probe Resource Center is a centralized desktop application that contains all of Keysight's oscilloscope probe documentation: data sheets, user guides, response plots, SPICE models, and more. The Probe Resource Center will also alert you when it has been updated (either because of new probes or changes to documentation), and all it takes is a simple click of a button to download and install the most up-to-date version.

To install the Keysight Probe Resource Center application:

  1. Install Adobe AIR.

    Adobe AIR must be installed before you can install the Keysight Probe Resource Center AIR Help file. You can download Adobe AIR from www.adobe.com.

  2. Get the latest version of the Keysight Probe Resource Center AIR Help file (PRC.air) from www.keysight.com/find/prc and install it.

Once installed, you can launch the Probe Resource Center from the Windows Start menu by choosing All Programs > Keysight Probe Resource Center.
